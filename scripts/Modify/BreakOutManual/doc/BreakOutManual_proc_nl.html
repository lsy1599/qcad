1. Selecteer het object dat u wilt opsplitsen.
  2. Klik op de eerste punt waarop het object moet worden verdeeld.
  3. Als de punt niet op het object ligt, wordt de dichtstbijzijnde punt op het object gebruikt.
  4. Klik op de tweede punt waarop het object moet worden verdeeld.
  5. Het segment tussen de twee punten wordt verwijderd als de optie "Segment verwijderen" in de opties werkbalk actief is. In het andere geval wordt het object slechts gescheiden op de gegeven punten.
